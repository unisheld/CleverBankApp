# CleverBankApp

Консольное приложение CleverBankApp для Clever-Bank.

## Описание

Приложение предоставляет следующие функции:

- Пополнение и снятие средств.
- Перевод средств.
- Печать чека после каждой операции.
- Начисление процентов.
- Формирование выписки в формате PDF.

## Требования

Для сборки и запуска проекта CleverBankApp вам потребуются следующие компоненты и зависимости:

- Java Development Kit (JDK) 8 или новее.
- База данных PostgreSQL.
- Библиотека iText версии 5.5.13 для работы с PDF.
- Библиотека SnakeYAML версии 2.0 для работы с YAML файлами.
- Библиотека Lombok версии 1.18.22 для упрощения написания кода.
- Фреймворк JUnit Jupiter версии 5.9.2 для модульного тестирования.
- Фреймворк Mockito версии 5.2.0 для создания заглушек и мок-объектов в тестах.

## Установка

1. Настройте соединение с базой данных:
   - В папке resources проекта найдите файл config.yml.
   - Внесите необходимые настройки соединения с базой данных, включая URL, имя пользователя и пароль.

2. Загрузите схему базы данных:
   - В папке resources проекта найдите файл cleverbank.sql.
   - Запустите этот SQL-файл для создания необходимых таблиц и записей.

3. Сборка проекта:
   - Вы можете собрать проект с помощью вашей IDE или с помощью сборочной системы Gradle. Проект уже содержит файл build.gradle для этой цели.

4. Запуск проекта:
   - После успешной сборки вы можете запустить приложение CleverBankApp для начала использования.

## Описание файлов

- `Main` - Главный класс для запуска приложения.
- `models` - Пакет содержит модели данных:
  - `Account` - Модель банковского счета.
  - `Bank` - Модель банка.
  - `Transaction` - Модель транзакции.
  - `TransactionType` - Модель типа транзакции.
  - `Client` - Модель клиента.
- `database` - Пакет содержит классы для работы с базой данных:
  - `DatabaseConnector` - Класс для подключения к базе данных.
  - `DatabaseReader` - Класс для чтения данных из базы данных.
  - `DatabaseUpdater` - Класс для обновления данных в базе данных.
  - `TransactionDAO` - Класс доступа к данным транзакций.
  - `TransactionSaver` - Класс для сохранения транзакций.
- `exceptions` - Пакет содержит классы исключений:
  - `InsufficientFundsException` - Исключение, возникающее при недостаточных средствах на счете.
- `interfaces` - Пакет содержит интерфейсы сервисов:
  - `TransactionService` - Интерфейс сервиса пополения и снятия средств.
  - `TransferService` - Интерфейс сервиса перевода средств.
- `services` - Пакет содержит сервисы приложения:
  - `CheckGeneratorService` - Сервис для генерации чеков.
  - `DepositService` - Сервис для внесения депозита.
  - `InterestCalculatorService` - Сервис для расчета процентов.
  - `StatementGeneratorService` - Сервис для генерации выписок.
  - `TransferService` - Сервис для выполнения переводов.
  - `WithdrawService` - Сервис для снятия средств.
- `utils` - Пакет содержит вспомогательные утилиты:
  - `AppConfig` - Конфигурационный класс.
- `resources` - Директория с ресурсами проекта:
  - `config.yml` - Файл конфигурации для настройки соединения с базой данных.
  - `cleverbank.sql` - Схема базы данных.
  - `Roboto-Light.ttf` - Шрифт для генерации PDF-документов.
